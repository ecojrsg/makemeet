title: Primer Ingreso y Autenticación
---

## 1) Abrir la aplicación

Ingresa a la URL donde está desplegada tu instancia.

## 2) Login

Puedes iniciar sesión con:

- Email/contraseña
- Google (si está habilitado)

## 3) Si aparece pantalla de setup

La instancia aún no está lista o faltan migraciones. Contacta al administrador y comparte el estado mostrado en `/setup`.
