title: Mejorar Texto con IA
---

## Requisitos

Configurar una API key en `/setup`:

- Temporal (solo sesión)
- Guardada (persistente)

## Uso

En campos de resumen/experiencia/educación, usa acciones de mejora de IA para refinar el texto.

## Prioridad de key

Si existe key temporal, se usa antes que la key guardada activa.
