title: Cambiar Plantilla
---

MakeMeEt incluye 4 plantillas:

- Moderna
- Clásica
- Minimalista
- Creativa

Puedes cambiar de plantilla desde el panel lateral sin perder tu contenido.

## Para contribuidores

Si necesitas crear una plantilla nueva o entender cómo funciona la exportación, consulta la guía técnica [Crear Plantillas de CV](/implementacion/crear-plantillas-cv).
