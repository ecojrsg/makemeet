id: troubleshooting
title: Solución de Problemas
---

## No puedo iniciar sesión

- Verifica que el proveedor de auth esté habilitado en Supabase.
- Revisa `VITE_AUTH_PROVIDERS` en entorno.

## No funciona IA

- Verifica que exista API key temporal o guardada activa.
- Usa la opción **Probar** en el panel de API keys.

## Error al exportar

- Intenta con otra plantilla o recarga la página.
- Si persiste, revisa consola del navegador.

## Setup no marca listo

- Reporta al administrador las tablas faltantes mostradas en `/setup`.
