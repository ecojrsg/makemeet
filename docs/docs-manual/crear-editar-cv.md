## Crear CV

1. Haz clic en **Nuevo CV**.
2. Completa datos personales.
3. Agrega experiencia, educación, skills e idiomas.

## Edición en tiempo real

Mientras editas, la vista previa se actualiza automáticamente.

## Integración GitHub (opcional)

Si agregas usuario de GitHub, la app puede usar perfil y repos como contexto.
