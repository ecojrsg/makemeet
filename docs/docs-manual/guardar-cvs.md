title: Guardar y Actualizar CVs
---

## Guardar

- Usa el botón **Guardar** para crear un CV nuevo con nombre y etiquetas.

## Actualizar

- Si ya cargaste un CV existente, el botón cambia a **Actualizar**.

## Organización

Puedes tener varios CVs y alternar entre ellos desde la barra lateral.
